Funcionalidades completas para gestão de contact center

Dashboard Executivo

Visão geral consolidada com métricas principais, KPIs e indicadores de performance em tempo real.

  • Métricas de HC total e por operação
  • Indicadores de SLA e ocupação
  • Gráficos de tendências e comparativos
  • Alertas e notificações importantes

Cálculo de Necessidade de HC

Algoritmo Erlang C para dimensionamento preciso baseado em volume, TMI e metas de SLA.

  • Cálculo automático por intervalo de tempo
  • Consideração de improdutividade
  • Simulação de diferentes cenários
  • Otimização de recursos

Comparação Escala vs Necessidade

Análise comparativa entre a escala atual e a necessidade calculada para identificar gaps.

  • Visualização de déficit/superávit por período
  • Recomendações de ajustes
  • Impacto no SLA
  • Análise de custos

Taxa de Ocupação

Monitoramento da taxa de ocupação dos recursos para otimização da produtividade.

  • Ocupação por agente e equipe
  • Tendências históricas
  • Benchmarks e metas
  • Identificação de oportunidades

Gestão de Horas Extras

Controle e análise de horas extras com impacto financeiro e operacional.

  • Planejamento de horas extras
  • Análise de custos
  • Alternativas de cobertura
  • Relatórios gerenciais

Módulo de Treinamentos

Gestão de treinamentos com impacto na disponibilidade de recursos.

  • Planejamento de treinamentos
  • Impacto na operação
  • Cronograma otimizado
  • Acompanhamento de progresso

Gestão Completa de Escalas

Criação, edição e otimização de escalas respeitando regras DSR e preferências.

  • Escalas automáticas e manuais
  • Regras DSR (75% domingo, 125% folgas)
  • Turnos customizáveis
  • Rotação de folgas

Performance Analytics

Análises avançadas de performance com múltiplas dimensões e indicadores.

  • HCs, volume e AHT/TMI
  • Nível de serviço (NS)
  • Taxa de improdutividade
  • Chamadas no SLA

Como Executar

Guia passo a passo para rodar o projeto localmente

Setup Automático (Recomendado)

# 1. Setup inicial das dependências
.\setup-simple.bat

# 2. Configurar banco e dados iniciais  
.\setup-database.bat

# 3. Iniciar backend
.\start-backend.bat

# 4. Iniciar frontend (novo terminal)
cd packages\frontend
npm run dev

Setup Manual

# Backend
cd packages\backend
npm install
npm run dev

# Frontend (novo terminal)
cd packages\frontend  
npm install
npm run dev

URLs de Acesso

  • Frontend: http://localhost:3000
  • Backend API: http://localhost:3001
  • Health Check: http://localhost:3001/health

Banco de Dados

  • Tipo: SQLite (dev.db)
  • ORM: Prisma
  • Seed: Dados de teste incluídos

Requisitos

  • Node.js: 18+ (atual: 22.16.0)
  • NPM: 10+
  • OS: Windows/Linux/macOS
